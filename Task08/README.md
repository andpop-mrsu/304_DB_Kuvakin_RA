# Система управления мастерами СТО

Веб-приложение для управления мастерами сервисной станции технического обслуживания.

## Требования

- PHP 7.4 или выше
- Расширение PDO с поддержкой SQLite
- Веб-сервер (Apache, Nginx или встроенный сервер PHP)

## Установка и запуск

### 1. Инициализация базы данных

Перед первым запуском необходимо инициализировать базу данных:

```bash
cd php/rk/Task08
php init_db.php
```

Это создаст файл `data/sto.db` с необходимыми таблицами и тестовыми данными.

### 2. Запуск веб-сервера

#### Вариант 1: Встроенный сервер PHP (рекомендуется для разработки)

```bash
cd php/rk/Task08
php -S localhost:3000 -t public
```

Затем откройте в браузере: `http://localhost:3000` или `http://localhost:3000/index.php`

#### Вариант 2: Настройка Apache/Nginx

Настройте виртуальный хост так, чтобы корневая директория указывала на `php/rk/Task08/public`

### 3. Проверка работы

После запуска вы должны увидеть страницу "Список мастеров СТО" с таблицей мастеров.

## Структура проекта

```
Task08/
├── config.php          # Конфигурация подключения к БД
├── init_db.php         # Скрипт инициализации базы данных
├── data/
│   └── sto.db         # База данных SQLite
├── public/
│   ├── index.php      # Главная страница (список мастеров)
│   ├── style.css      # Стили
│   ├── mechanic_*.php # Управление мастерами
│   ├── schedule_*.php # Управление графиком работы
│   └── work_log_*.php # Управление выполненными работами
└── README.md          # Этот файл
```

## Возможные проблемы

### Ошибка "База данных не найдена"

Убедитесь, что вы запустили `php init_db.php` перед первым запуском.

### Ошибка "PDO extension not found"

Установите расширение PDO для SQLite в PHP. В Windows это обычно включено по умолчанию.

### Ошибка "Permission denied"

Убедитесь, что папка `data/` имеет права на запись.

## Функционал

- Просмотр списка мастеров
- Добавление, редактирование и удаление мастеров
- Просмотр и управление графиком работы мастера
- Просмотр и управление выполненными работами мастера
